# PyCache

simple key:value server written by **Python**


## run server
```commandline
python -m pycache.server
```

or

```python
from pycache.server import start_server

start_server()
```


### set custom Host and Port

set an environment variables
```
HOST = your ip (like 127.0.0.1, 0.0.0.0)
PORT = your port
```
